    // Logique répétée...
  }

Problèmes :

  • ✗ Code dupliqué sur 20+ pages
  • ✗ Modification = Éditer tous les fichiers
  • ✗ Risque d'incohérence visuelle
  • ✗ Maintenance complexe et chronophage
  • ✗ Pas de navigation centrale Desktop
  • ✗ Design non unifié

APRÈS

📄 Code HTML (chaque page)
<head>
  <!-- Autres scripts -->
  
  <script src="assets/js/components/header.js"></script>
</head>

<body>
  <div id="header-placeholder"></div>
  
  <main>
    <!-- Votre contenu -->
  </main>
</body>

Avantages :

  • ✓ Un seul fichier JS - Architecture modulaire
  • ✓ Modification unique - 1 fichier à éditer
  • ✓ Cohérence garantie - Partout identique
  • ✓ Maintenance facile - Gain de temps énorme
  • ✓ Navigation centrale - Style Apple Desktop
  • ✓ Responsive parfait - Mobile & Desktop

⚡ Fonctionnalités automatiques

🛒 Compteur de panier

Lecture automatique du localStorage, mise à jour en temps réel, badge animé.

🔐 Authentification

Détection Firebase, affichage conditionnel connexion/déconnexion.

📜 Effet scroll

Ombre élégante au scroll > 50px, padding dynamique, opacité ajustée.

📱 Menu mobile

Drawer animé, overlay blur, fermeture automatique, gestion clavier.

🎨 Animations

Transitions fluides, cubic-bezier, hover effects, micro-interactions.

🔄 Auto-inject

Injection automatique au DOMContentLoaded, aucune action manuelle requise.
